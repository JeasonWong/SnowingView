## What's SnowingView ?
It's a dynamic wallpaper with snowing fall down effect and accelerometer sensor. 

## Demo
![SnowingView](./design/snowingwallpaper.gif)

## 中文介绍
- 三种方式实现雪花掉落效果（原作者的View，我补充的SurfaceView以及WallpaperService）
- 拥有加速度传感器，效果更酷
